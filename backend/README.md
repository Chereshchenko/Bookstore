# BOOK STORE
BookStore – онлайн-магазин книг с возможностью просмотра каталога, оформления заказов и оставления отзывов.

Проект представляет собой RESTful API для книжного магазина, реализованный на Flask. Основные возможности:

📚 Просмотр каталога книг с фильтрацией по жанру, цене и поиском;

🛒 Добавление книг в корзину и оформление заказов;

📦 Поддержка разных способов доставки (самовывоз, курьер);

⭐ Оставление отзывов и рейтингов;

🔍 Сортировка книг по рейтингу, цене и дате;

📝 Личный кабинет пользователя с историей заказов.

API предназначено для интеграции с фронтендом (React/Vue) или мобильным приложением.

В проекте используются следующие технологии:
1. Python 3.10+;
2. Flask (веб-фреймворк);
3. Flask-Login (аутентификация);
4. SQLAlchemy (ORM для работы с БД);
5. PostgreSQL (реляционная БД);

Для запуска проекта, необходимо:

1) Установить зависимости. pip install -r requirements.txt.

2) Настроить базу данных. Создать файл .env в корне проекта и прописать DATABASE_URL и SECRET_KEY. 

3) Запусть сервер. flask run --host=0.0.0.0 --port=5000
