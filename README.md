## 📚 Интернет-магазин книг
Веб-приложение для покупки книг с возможностью просмотра каталога, регистрации, добавления в корзину и оформления заказов.
Проект реализован на Flask с использованием SQLAlchemy и шаблонизатора Jinja2. Для авторизации используется Flask-Login.

## 🎯 Цели и задачи
Создать полнофункциональный интернет-магазин, который позволяет:
- просматривать каталог книг с разбивкой по жанрам;
- искать книги по названию или автору;
- регистрироваться и входить в систему;
- добавлять книги в корзину и управлять её содержимым;
- оформлять заказы с выбором способа доставки;
- просматривать историю заказов;
- оставлять отзывы и оценки на книги (для авторизованных пользователей).

## 🧱 Основной функционал
## 👤 Пользователи (неавторизованные)
- Главная страница с ТОП-3 книг недели, жанрами и поиском.
- Просмотр каталога по жанрам.
- Просмотр страницы книги (название, автор, цена, описание, обложка, рейтинг).

## 👤 Авторизованные пользователи
- Всё, что доступно неавторизованным.
- Добавление книг в корзину.
- Управление корзиной (изменение количества, удаление).
- Оформление заказа (выбор доставки: самовывоз или доставка по адресу).
- История заказов с деталями и статусами.
- Возможность оставлять отзывы и ставить оценки книгам.
- Выход из аккаунта с подтверждением.

## 🔐 Аутентификация
- Регистрация с полями: имя, email, телефон, пароль.
- Подтверждение регистрации через код (имитация — без реальной СМС).
- Вход по email и паролю.

- Flask

Инструменты:
